# Flaer Legal Documents

This repository contains all official legal documents for Flaer, including the **Privacy Policy**, **Terms of Service**, and **Terms of Monetization**.  
Each document is versioned, immutable once published, and stored in both **DOCX** (for legal review) and **Markdown** (for website publication).  
Optional summaries ("overviews" or "one-pagers") may accompany each version.
